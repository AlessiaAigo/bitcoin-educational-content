---
term: OP_GREATERTHANOREQUAL (0XA2)
---

Vergleicht die zwei obersten Elemente auf dem Stapel und prüft, ob das erste Element größer oder gleich dem zweiten ist. Wenn das erste Element größer oder gleich dem zweiten ist, wird `1` (wahr) auf den Stapel gelegt; andernfalls wird `0` (falsch) darauf gelegt.
