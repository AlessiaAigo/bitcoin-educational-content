---
term: BITCOIN.CONF
---

File di configurazione utilizzato per personalizzare il funzionamento di un nodo Bitcoin che esegue il client Bitcoin Core. Situato nella directory dei dati di Bitcoin Core, questo file di testo consente agli operatori del nodo di specificare vari parametri e opzioni che influenzano il comportamento del nodo.
