---
Begriff: SDK (SOFTWARE DEVELOPMENT KIT)
---

Ein Satz von Softwarewerkzeugen, die die notwendigen Ressourcen für Entwickler bereitstellen, um Anwendungen auf einer spezifischen Plattform zu erstellen. Ein SDK umfasst Bibliotheken, Entwicklungshandbücher, Codebeispiele und Kompilierungsprozesse. SDKs erleichtern und beschleunigen die Entwicklung, indem sie wiederverwendbare Module anbieten. Bei Bitcoin gibt es das BDK (*Bitcoin Dev Kit*) und das LDK (*Lightning Dev Kit*).

> ► Auf Englisch werden SDKs manchmal auch als "devkit" bezeichnet. Für weitere Informationen siehe die Definitionen von [**BDK**](/dictionnaire/B.md#bdk-bitcoin-dev-kit) und [**LDK**](/dictionnaire/L.md#ldk-lightning-dev-kit).
